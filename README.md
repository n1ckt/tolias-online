# tolias.online

Personal website for Nick Tolias.

## Local preview

Open `index.html` directly in your browser, or run a tiny local server:

```bash
cd tolias-online
python3 -m http.server 8000
# Visit http://localhost:8000
```

## Files

- `index.html` — the whole site (single page, no build step)
- `assets/profile.jpg` — profile photo
- `CNAME` — custom domain for GitHub Pages (not needed for Cloudflare Pages)

## Deploy to Cloudflare Pages

See `DEPLOY.md` for step-by-step instructions.

## Edit the content

Everything is in `index.html`. Open it in any text editor, change the text, save, refresh the browser. No build step.
